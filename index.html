Claude란 무엇인가요?

Claude는 Anthropic이 개발한 대규모 언어 모델(LLM) 기반의 AI 어시스턴트입니다. 안전하고 유용하며 정직한 AI를 목표로 설 계되었으며, 웹 채팅, API, 그리고 Claude Code와 같은 다양한 인터페이스를 통해 이용할 수 있습 니다. 현재 최신 모델은 Claude Opus 4.6이며, Claude 4 모델 패밀리에 속합니다.

뛰어난 추론 능력

복잡한 논리적 문제, 수학, 코딩 등 다 양한 분야에서 깊이 있는 사고를 제공합니다.

🌍

다국어 지원

한국어를 포함한 다양한 언어로 자연스 러운 대화와 번역이 가능합니다.

🔒

안전성 최우선

📄

긴 문서 처리

자주 묻는 질문

Claude는 어디서 사용할 수 있나요? +
웹·모바일·데스크톱 채팅 인터페이스, API, 그리고 터미널 기반의 Claude Code를 통해 사용할 수 있습니다.
Claude는 이전 대화를 기억하나 요? +
아니요. Claude는 대화 간 정보를 보존하지 않습니다. 각 대화는 독립적 으로 처리됩니다.
Claude Code란 무엇인가요? +
Claude Code는 개발자가 터미널에서 직접 Claude에게 코딩 작업을 위임할 수 있는 에이전틱 코딩 도구입니다. 자세한 내용은 공식 문서를 참고하세요.
API는 어떻게 사용하나요? +
Anthropic API 문서(docs.anthropic.com)에서 자세한 가이드를 확인하실 수 있습니다.
